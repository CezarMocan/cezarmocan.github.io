World Upstream: Arab Bank Emerging Artist Prize

2023-2024

digital simulation

Greetings ✨

This page is part of my submission for the Arab Bank Emerging Artist prize.

World Upstream is a real-time simulation of infinite duration, which exists as a .exe software. Since this file format was not an option in the submission form, I'm adding a few resources here.

Video Documentation
On the right side you can watch a 3 minute long trailer of the project, as well as two screen recordings of the simulation: a short one (7 minutes) and a long one (3 hours and 19 minutes.)

The Software
You can download the simulation here. The software only runs on Windows and requires a dedicated, recent graphics card (NVIDIA RTX 3060 or equivalent,) a standard gaming setup.

Other Materials
You can find a PDF presentation of the work here.

A note on minting
While I've minted other works from this series as NFTs, I haven't done it for the core simulation. I'm excited to find a way for software like this to meaningfully exist as an NFT. Currently the work exists in the contemporary art market, as an edition of 3 + 2AP.

Thank you for your consideration,
c

The 3h19 recording of the simulation is currently on view on common.garden, part of Down the Silicon Meadow, an exhibition curated by OFFICE IMPART and Blueshift Gallery.
